# Worthy – Personalized Beauty Brand Website

## Project Overview

Worthy is a React-based web application developed for a factory that produces personalized beauty and self-care products such as shampoos, make-up products and self-care gift boxes.

The purpose of the website is to allow users to create their own beauty brand by customizing products directly through the platform. Users can choose product types, select packaging designs, add notes and specifications, manage quantities, and create personalized orders.

The application was created as part of a React course project to demonstrate practical knowledge of React development, component structure, state management, routing, forms, CRUD operations, and UI design.

---

## Features

- Product listing page
- Product customization
- Shopping cart system
- Create, edit and delete orders
- Admin dashboard
- Login and signup system
- LocalStorage data persistence
- Dynamic UI previews
- Responsive and organized interface
- React Router navigation
